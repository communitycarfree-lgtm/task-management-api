🚀 Quick Start

Get your environment running in under two minutes.

1. Clone

Clone the repository and ensure you have the .NET 9 SDK installed.

git clone <repo-url>

2. Run

Start the app — EF Core will auto-apply migrations on first launch.

dotnet run

3. Explore

Open the interactive Swagger UI to call your first endpoint.

/swagger

Overview

A production-ready REST API built on .NET 9, ASP.NET Core, and PostgreSQL following Clean Architecture principles.

Base URL
/api  —  All endpoints are prefixed with this path.

What the API provides

Project Management

Create, update, and archive projects with role-based team membership controls.

Task Tracking

Granular task lifecycle — priorities, assignees, deadlines, and dependency chains.

Real-time Sync

SignalR hub pushes live updates to connected clients without polling.

JWT Auth & RBAC

Secure Bearer token auth with Admin / Manager / Member role hierarchy.

First Request

Three requests to get you from zero to your first task.

Register

HTTP 
POST /api/auth/register
Content-Type: application/json

{
  "email": "user@example.com",
  "password": "SecurePass123!",
  "firstName": "Jane",
  "lastName": "Doe"
}

Create a Project

HTTP 
POST /api/projects
Authorization: Bearer {your-token}
Content-Type: application/json

{
  "name": "My First Project",
  "description": "A project for managing tasks"
}

Add a Task

HTTP 
POST /api/tasks
Authorization: Bearer {your-token}
Content-Type: application/json

{
  "projectId": "your-project-id",
  "title": "Implement login screen",
  "priority": "High",
  "dueDate": "2026-07-01"
}

Authentication

All protected endpoints require a JWT Bearer token in the Authorization header.

Token lifetime
Access tokens expire after 60 minutes. Use POST /api/auth/refresh with your refresh token to obtain a new pair without re-entering credentials.

Login

HTTP 
POST /api/auth/login
Content-Type: application/json

{
  "email": "user@example.com",
  "password": "SecurePass123!"
}
