Projects Endpoints

Endpoints for managing project lifecycle and team collaboration.

Project Management

GET /api/projects
List all projects with pagination and filtering
POST /api/projects
Create a new project workspace
GET /api/projects/{id}
Get detailed information about a specific project
PUT /api/projects/{id}
Update project metadata and settings
DELETE /api/projects/{id}
Archive or soft-delete a project

Team Collaboration

POST /api/projects/{id}/members
Invite a team member to the project
DELETE /api/projects/{id}/members/{userId}
Remove a member from project access

Tasks Endpoints

Fine-grained control over task creation, assignment, and tracking.

Task Operations

GET /api/tasks
Query tasks across all projects
POST /api/tasks
Create a new task within a project context
PUT /api/tasks/{id}
Update task details, description, and metadata
DELETE /api/tasks/{id}
Remove a task from the system
PUT /api/tasks/{id}/status
Transition task to a new lifecycle state
PUT /api/tasks/{id}/assignee
Update task ownership/assignee

Users Endpoints

Manage user profiles, security settings, and administrative roles.

Profile Management

GET /api/users/{id}
Retrieve a user profile with roles
PUT /api/users/{id}
Update user's full name and profile details
PUT /api/users/{id}/password
Securely change account password

Administration

GET /api/users
List all registered users (Admin only)
POST /api/users/{id}/roles
Assign administrative roles to a user

Notifications Endpoints

Manage your personal notification feed and alert status.

Alert Inbox

GET /api/notifications
Get paginated alerts for the current user
PUT /api/notifications/{id}/read
Mark a specific alert as read
DELETE /api/notifications/{id}
Permanently remove an alert from the inbox

Authentication

Secure access control using JWT Bearer tokens.

Identity Provider

POST /api/auth/register
Create a new platform identity
POST /api/auth/login
Exchange credentials for a JWT access token
POST /api/auth/refresh
Issue a new token using a valid refresh token
