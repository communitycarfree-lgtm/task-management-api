🚀 Core Capabilities

A full-featured engine for professional task and project management at any scale.

Task Dependency Graph

Model real-world workflows where task B cannot start until task A is complete. Circular dependencies are automatically detected and rejected.

  • Blocking & blocked-by links
  • Cycle detection on write
  • Cascade status propagation

Time Tracking

Precise per-task time logging with estimated vs. actual hours — giving project managers accurate capacity data for planning.

  • Start / stop time entries
  • Estimated vs. actual delta
  • Per-user & per-project rollup

Status State Machine

Tasks move through a defined lifecycle — Todo → InProgress → InReview → Done. Each transition is validated server-side; invalid jumps return a clear error.

  • Validated transitions only
  • Backward re-open allowed
  • Transition timestamp recorded

Filtering & Pagination

Every list endpoint supports rich server-side filtering — by status, priority, assignee, date range — with offset-based pagination and configurable page sizes.

  • Multi-field filter params
  • Configurable page size
  • Consistent sort ordering

🛡️ Enterprise Security

Industry-standard protocols protecting every layer of your data.

JWT Authentication

State-of-the-art token authentication with HMAC-SHA256 signature verification and configurable expiry.

Scheme: Bearer JWT

Role-Based Access

Granular RBAC controls ensure users can only access resources appropriate for their permission level.

Admin | Manager | Member

Audit Logging

Every state-changing operation is recorded with actor identity, timestamp, and delta for compliance.

Immutable audit trail

Soft Delete

Resources are never immediately purged — they are archived with restoration capability built in.

DeletedAt timestamp pattern
Password Security
All passwords are hashed with BCrypt (cost factor 12). Plain-text credentials are never stored or logged at any layer.

🤝 Team Collaboration

Built for teams — from a two-person startup to an enterprise engineering org.

Membership Management

Invite, remove, and manage project members with distinct role assignments per project.

Task Assignment

Assign and reassign tasks to any project member with full audit trail of ownership changes.

Notification Inbox

Every team event — assignment, status change, comment — triggers targeted notifications.

Activity Visibility

See exactly who did what and when, across every project and task in the system.

⚡ Real-time Connectivity

Stay in sync with your team without polling or page refreshes.

SignalR WebSocket Hub
All task and project events are broadcast to connected clients in real time via ASP.NET Core SignalR. Supports WebSockets with Server-Sent Events as fallback.

What gets pushed live

Live Collaboration

See in real time who is working on what — task status changes appear instantly across all connected clients.

Instant Notifications

Alert delivery when a task is assigned, a deadline approaches, or a teammate comments on your work.

Multi-client State Sync

Web, mobile, and desktop clients all receive the same events simultaneously from the hub.

Automatic Reconnection

SignalR automatically re-establishes dropped connections and replays missed events on reconnect.

🏎️ High Performance

Engineered for speed — built to scale for global organizations under peak load.

Redis Caching

Distributed cache layer provides sub-millisecond retrieval for high-frequency reads like project lists and user lookups.

Response Compression

Gzip and Brotli compression for all API responses, reducing payload sizes by up to 75% on JSON-heavy endpoints.

Optimized EF Core

Compiled queries, AsNoTracking reads, and split queries prevent N+1 problems and minimize database round-trips.

Safe Pagination

All list endpoints use offset-based pagination with configurable page sizes — predictable, consistent retrieval at any dataset scale.
